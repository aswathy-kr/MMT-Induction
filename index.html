Where To Start ?

There are A LOT of languages and tools you can use to build for the web. All of them have their benefits, and are maintained by tons of super smart people. While this is great for pushing the web forward, it’s confusing and overwhelming to start.

Learn HTML, CSS and Javascript exclusively.

While HTML provides the structure for a website and CSS allows you to control the presentation of a site, the JavaScript programming language gives you the tools that you need to alter the behavior of different elements that are found on a website page. If you know HTML, CSS, and Javascript on a deep level, you will be able to create pretty much anything on the web.

Tools To Use

Next up, download a good software to use as your code editor. Below are a few :

  • VSCode
  • Atom.io
  • Sublime Text

After that you’ll need a terminal to run commands and do various other things. MacOs already has a inbuilt Terminal app. Click the Launchpad icon in the Dock, type Terminal in the search field, then click Terminal. For Windows users, install Unix-like terminals that execute bash commands. Check out this.

Lastly, you need a browser to test your website in, so grab the latest version of Google Chrome.

HTML stands for Hypertext Markup Language. It is a standard markup language for building static web applications and websites. This is a relatively simplistic yet powerful programming language that allows web developers and website owners to create the structure of their websites.

CSS stands for Cascading Style Sheets. It is a style sheet language that’s focused almost entirely on improving the presentation of HTML elements. While CSS is a static programming language, it can be used to make your website appear visually pleasing and modern.
