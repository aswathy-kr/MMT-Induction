Introduction

First, let me introduce myself. My name is Aswathy K Raj and here I intend on sharing how I started my Web Development journey with a glimpse at some of my projects and courses/resources that I took help from during the process and a few tips from my side if you are aiming towards the Web Development domain.

Background

I graduated with a degree in Computer Science back in 2012 and immediately landed a job in the Services industry. I had worked as a Technical Support Engineer and later on as a Platform Engineer for a few years. Yet I had this innate desire to try out web development as I was fascinated by the way websites are designed. So I took up a course on Full Stack Web Development which has helped me to make a start in this journey. That was a huge milestone for me as I had no real experience doing web development.

Where To Start ?

There are A LOT of languages and tools you can use to build for the web. All of them have their benefits, and are maintained by tons of super smart people. While this is great for pushing the web forward, it’s confusing and overwhelming to start.

Learn HTML, CSS and Javascript exclusively.

If you know HTML, CSS, and Javascript on a deep level, you will be able to create pretty much anything on the web.
